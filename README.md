# Capresta
### Offline Voice Recognition Hardware for Makers

Capresta is an ESP32-S3 based offline voice recognition 
hardware module developed by Caprester (India). No cloud. 
No internet. Just fast, local voice control.

---

##  Features

-  **Fully Offline** — All processing on-device, 
  zero cloud dependency
-  **Espressif AFE Framework** — Utilizing Espressif's 
  Audio Front-End for highly accurate voice detection
-  **UART Configuration** — Customize voice commands 
  and map GPIO outputs via simple UART commands
-  **GPIO Output Mapping** — Map any GPIO output to 
  any command ID via UART
-  **Easy Integration** — Works seamlessly with 
  Arduino, Raspberry Pi, and custom PCB projects

---

##  Hardware

- **MCU:** ESP32-S3
- **Framework:** ESP-SR / ESP-IDF
- **Status:** Prototyping & Active Testing phase
  - Multi-PCB testing rig fully functional
  - Successfully validated with default wake word
